# Concert City V3

## Setup

1. Copy `.env.example` to a new file named `.env`.
2. Put your Ticketmaster Discovery API key after:
   `VITE_TICKETMASTER_API_KEY=`
3. Run:

```powershell
npm.cmd install
npm.cmd run dev
```

Open `http://localhost:5173`.

Favorites are saved in the browser with localStorage.
